# DropTower

## 概要
皿の上にランダムなブロックを積み上げていくゲーム  
ブロックが皿から落ちたらゲームオーバーとなり、その時の積み上げた高さがポイントとなる

## 内容
- 皿の上にランダム形状のブロックを積み上げる
- ブロックを落とす水平位置はプレイヤーが自由に指定できる
- ブロックが皿の外に落ちた時点でゲームオーバー　
- ゲームオーバー時の「皿の表面から最も高いブロックの頂点までの高さ」がスコア
- スコアは名前、日時と共に上位10位のランキングに記録される
- ゲーム中はいつでもポーズ（一時停止）でき、トップ画面に戻れる

## 画面構成
```
トップ画面
 ├─ [スタートボタン] ─> ゲーム画面
 └─ [スコアボタン] ─> スコア画面

ゲーム画面
 └─ Pose中 ─> [トップに戻るボタン] ─> トップ画面

スコア画面
 └─ [戻るボタン] ─> トップ画面
```
